# 安全政策

## 支持范围

目前只对 Chrome Web Store 最新公开版本和 `main` 分支提供安全修复。

## 私密报告

请不要用公开 issue 披露可被利用的漏洞。使用 GitHub 的 [Private vulnerability reporting](https://github.com/NorthGateChat/agora-philosophy/security/advisories/new)，包括：

- 受影响版本和环境；
- 可重复的步骤或最小示例；
- 可能影响；
- 已知的缓解方式。

维护者会尽量在 7 天内确认收到报告。请给项目合理的修复与发布窗口，再公开技术细节。

内容归属、翻译和史实问题不属于安全漏洞，请使用内容纠错模板。
