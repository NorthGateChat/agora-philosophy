# AGORA｜哲学新标签页 隐私说明

**生效日期：2026-07-23**

**维护者：NorthGateChat**

“AGORA｜哲学新标签页”（以下简称“AGORA”）希望用尽可能少的数据完成一件事：在新标签页呈现哲学命题、双语文本、解释与思想关联。当前版本无需账号、无需任何 Chrome 权限，也不包含广告或用户行为追踪。

## 我们不收集什么

当前版本不收集、上传、出售或共享任何个人数据，包括但不限于：

- 姓名、邮箱、账号、设备标识符或其他身份信息；
- 浏览历史、访问网址、搜索词、网页内容或下载记录；
- 精确或大致位置；
- 键盘输入、剪贴板历史或其他应用中的内容；
- 使用频率、点击行为、崩溃遥测或广告画像。

AGORA 不使用第三方分析 SDK、广告 SDK、追踪像素或远程日志服务，也不为用户建立跨网站或跨设备画像。

## 浏览器权限

当前版本申请的 Chrome 权限数量为 **0**。它只通过清单中的新标签页覆盖能力显示自己的页面，不读取其他标签页、网站或浏览器历史。

如果未来某个功能确实需要新增权限，我们会先更新本隐私说明和 Chrome Web Store 的数据披露，并在产品中解释用途；不会为尚未实现的功能预先申请权限。

## 仅保存在本机的数据

为了记住用户的选择，AGORA 使用扩展页面自己的 `localStorage` 保存：

- 主语言（简体中文、繁体中文或英文）；
- 中文句子是否同时显示英文；
- 视觉主题等界面偏好；
- 用户主动收藏的哲学命题。

这些内容只保存在当前浏览器的本地扩展存储中，不会由 AGORA 上传到服务器，也不会由 AGORA 在设备间同步。AGORA 没有服务器端账号系统，因此发布者无法查看或恢复这些本地选择。

用户可以通过以下方式删除本地数据：

1. 在 Chrome 中移除“AGORA”扩展；或
2. 在 Chrome 的扩展/站点数据设置中清除该扩展的存储数据。

删除本地数据后，语言、主题和收藏会恢复为初始状态。

## 用户主动触发的系统功能

分享图片由 AGORA 在用户的浏览器本地生成。打开分享弹窗、预览图片或切换图片版式不会上传图片，也不会向第三方发送数据。

只有当用户主动点击系统分享或某个第三方平台按钮时，当前分享文案、Chrome Web Store 商店链接以及用户明确选择分享的内容，才会交给浏览器、操作系统或对应的第三方平台：

- 使用系统分享时，浏览器或操作系统会显示分享面板；在环境支持的情况下，本地生成的分享图片也会作为文件交给用户选择的目标应用；
- 使用 X、微博、Facebook、LinkedIn 等平台按钮时，AGORA 会打开对应平台的分享页面，并按该平台支持的方式传递当前分享文案和/或商店链接；这些直接平台按钮不会上传或附带本地生成的分享图片；
- 用户如需在第三方平台使用分享图片，可以主动保存图片后自行上传，或在环境支持时通过系统分享发送；
- 用户主动选择“仅复制链接”或其他复制操作时，AGORA 只会把相应内容写入剪贴板。

AGORA 不会读取剪贴板或剪贴板历史，也不会接收用户的第三方平台账号、联系人、最终发布内容或分享结果。

用户选择的系统目标应用或第三方平台及其后续数据处理，受对应服务自身的隐私规则约束。

## 第三方共享、出售与广告

AGORA 不向第三方出售或共享用户数据，不使用用户数据投放个性化广告，也不进行信用评估、保险评估或其他敏感用途。由于当前版本不收集用户数据，也不存在由 AGORA 保存的数据留存期限。

## 儿童隐私

AGORA 不是面向儿童定向收集数据的服务。当前版本不收集任何用户的个人数据；如果产品未来增加账号、投稿或联网功能，会在上线前重新评估适用的儿童隐私义务。

## 隐私说明变更

如果产品功能或数据处理方式发生变化，我们会更新本页、生效日期以及 Chrome Web Store 中的隐私披露。重大变化会在合理可见的位置说明，不会用更新隐私说明来追认此前未披露的数据收集。
