# 品牌决定：AGORA

## 最终推荐

- 产品名：**AGORA**
- 类别名：**哲学新标签页**
- 完整中文名：**AGORA｜哲学新标签页**
- 英文商店名：**AGORA — Philosophy New Tab**
- 主 slogan：**一页，一种思想。**
- 传播口号：**不是金句，是思想的来路与去处。**
- 一句话利益点：**每开一页，换一种看世界的方法。**

Agora 是古希腊城邦的公共广场：人们在这里相遇、讨论、质问与辩论。它天然对应这款产品的目标——不把哲学封存在名言墙里，而是让不同年代、不同立场的观点重新相遇。

所有语言统一使用大写 `AGORA`，不另设中文译名或并列品牌。Chrome Web Store 的英文名称固定使用 `AGORA — Philosophy New Tab`。

## 为什么用“AGORA”

它直接来自西方哲学的文化语境，又不是某一位哲学家或某一个学派的专名。它既可以容纳苏格拉底式的追问，也可以容纳后世思想之间的继承、分歧和回应；作为界面角落里的品牌标识，也不会被误解成作者署名。

名称仍需在正式发布前完成商标、域名和应用商店重名检索；当前决定不构成法律意见。

## 传播层级

1. **识别产品**：AGORA｜哲学新标签页
2. **建立记忆**：一页，一种思想。
3. **解释差异**：不是金句，是思想的来路与去处。
4. **说明功能**：点开命题，继续阅读双语译写、解释、思想来处、同题异答和后续回响。

## 不使用“最全”

首发版本不使用“最全哲学家”“最全哲学思想”等绝对表述。当前内容的优势是观点之间可继续探索，而不是条目数量。更可靠的说法是：

- “一张可以继续展开的哲学手稿”
- “从一句命题，进入它的思想脉络”
- “持续扩充的哲学思想地图”

当哲学家覆盖、原典版本、引用定位和编辑规范都形成公开标准后，再用可验证的数量描述规模。

## 视觉提示

品牌符号是一枚留有缺口的圆：圆表示完整答案的诱惑，缺口表示继续追问的余地。中心点代表当前命题，金色小点代表即将进入的下一种思想。文字始终保持稳定，只有环境线条和节点做轻微运动。

## 配置位置

品牌名、版本、slogan、传播口号和多语言描述统一维护在项目根目录的 `brand.json`。扩展清单、网页元信息、分享标题与商店素材应从这里生成或读取，避免改名时遗漏。
