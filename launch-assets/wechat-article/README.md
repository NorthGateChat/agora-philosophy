## 发布建议

1. 公众号正文用 16px 左右字号、1.75—1.9 行距，段前后保留 8—12px。
2. 正文不要再额外加复杂模板；黑色小标题、象牙白图片和一条沙金分隔线即可。
3. 两张详情图建议分别单独占满正文宽度，不要左右拼接。
4. `07`、`08` 已按手机阅读重排，每张包含两条完整观点，不需要再次裁剪。
5. 外链受公众号限制时，可把 Chrome 商店和 GitHub 链接放在“阅读原文”，正文末尾保留项目名与可复制的短链接。

核心视觉来自 AGORA 自身：公众号封面采用纯黑白的古希腊 Agora 对谈场景，不重复文章标题；正文插画沿用象牙白、炭黑、沙金与代表“拒绝封闭答案”的断环。三张章节插画均为本次文章原创生成，没有使用希腊雕像、发光大脑、机器人头像等常见科技哲学意象。

第二张插画有意比另外两张更接近档案纸，以提示“思想谱系”的历史层次；正文中只出现一次，不作为全篇底色。

## 重新生成封面与长图

在项目根目录运行：

```bash
node launch-assets/wechat-article/source/render-assets.mjs
```

脚本会从保存在 `source/cover-agora-scene-original.png` 的高清原创图重建 900×383 黑白封面，同时重建两张截图长图；不会改动三张正文插画或六张真实产品截图。
